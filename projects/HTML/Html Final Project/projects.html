Project 1 - HTML in Action

Back to Projects

This project is about doing a quiz , which contains questions with their answers. This project it is done in VS Code




As you see there are the 10 questions, but the quiz contains total 20 questions. The first 10 questions are in the first page and the other questions till 20 are in the second page
In the end of the first page we see the link "Next Page" ,when we click it sends us to the second page.




We also can see that each question has the word "Answer". This is a link, that when we click it, it sends us to the right specific answer of the question. This action it is basically done by the help of ID,which defines a unique identifier.




So we clicked "Answer" for example question 14 and then it sended us to the answer of the 14th question and if we click again the "Back to question" , it sends us again in the second page of the specific question.

This was done with the semantic tags.

This project is about using semantic tags
The semantic HTML tags help the search engines and other user devices to determine the importance and context of web pages. The pages made with semantic elements are much easier to read. It has greater accessibility. It offers a better user experience.





As we see there is a difference between "Non-Semantic tags" and "Semantic Tags". Semantig tags include:

  • header
  • nav
  • aside
  • main
  • article
  • figure
  • p
  • section
  • footer
